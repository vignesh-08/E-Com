# AuraStyle - Responsive E-Commerce Product Listing Page

## Overview

AuraStyle is a fully responsive E-Commerce Product Listing application built using pure HTML, CSS, and JavaScript. The application allows users to browse products, search items, filter by category, sort products by price, and manage a shopping cart with persistent storage.

This project was developed as part of a Frontend Developer Assessment.

---

## Features

### Product Listing

* Displays 25 products across multiple categories
* Product cards include:

  * Product Image
  * Product Name
  * Product Price
  * Product Category

### Search Functionality

* Real-time product search
* Search products by name

### Category Filtering

* Filter products by category:

  * Electronics
  * Fashion
  * Footwear
  * Accessories
  * Home Decor

### Price Sorting

* Sort products by:

  * Price: Low to High
  * Price: High to Low

### Shopping Cart

* Add products to cart
* Update quantity
* Remove products
* Clear entire cart
* Dynamic cart item counter
* Cart subtotal calculation

### Responsive Design

* Mobile-first approach
* Tablet optimized layout
* Desktop optimized layout
* Responsive product grid

### Dark / Light Mode

* Theme toggle support
* User preference saved automatically

### Local Storage Support

* Cart data persists after page refresh
* Theme preference persists between sessions

### User Experience Enhancements

* Toast notifications
* Smooth animations and transitions
* Sticky header and filter section
* Empty state handling
* Accessible UI elements

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

No external frameworks or libraries were used.

## How to Run

### Method 1: Open Directly

1. Download or clone the repository.
2. Extract the ZIP file if applicable.
3. Open `index.html` in any modern web browser.

### Method 2: Using VS Code Live Server

1. Open project folder in Visual Studio Code.
2. Install the Live Server extension.
3. Right-click on `index.html`.
4. Click **Open with Live Server**.
